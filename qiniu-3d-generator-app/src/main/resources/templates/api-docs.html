/api/generation/text-to-3d
POST
根据文本描述生成3D模型。支持多种风格和质量设置，生成过程为异步处理。

请求参数

参数名 类型 必填 说明
prompt string 是 文本描述，用于生成3D模型
style string 否 生成风格：realistic, cartoon, anime
quality string 否 生成质量：low, medium, high
format string 否 输出格式：obj, fbx, glb 
curl -X POST "http://localhost:8091/api/generation/text-to-3d" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "一只可爱的小猫咪",
    "style": "realistic",
    "quality": "high",
    "format": "obj"
  }'
const response = await fetch('http://localhost:8091/api/generation/text-to-3d', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    prompt: '一只可爱的小猫咪',
    style: 'realistic',
    quality: 'high',
    format: 'obj'
  })
});

const result = await response.json();
console.log(result);
import requests

url = "http://localhost:8091/api/generation/text-to-3d"
data = {
    "prompt": "一只可爱的小猫咪",
    "style": "realistic",
    "quality": "high",
    "format": "obj"
}

response = requests.post(url, json=data)
result = response.json()
print(result)

响应示例

200 OK
{
  "code": "0000",
  "info": "success",
  "data": {
    "taskId": "task_123456789",
    "status": "processing",
    "estimatedTime": 300,
    "message": "任务已提交，正在生成中..."
  }
}
/api/generation/image-to-3d
POST
根据上传的图片生成3D模型。支持多种图片格式，自动识别物体轮廓并生成对应的3D模型。

请求参数

参数名 类型 必填 说明
image file 是 图片文件，支持jpg、png、webp格式
quality string 否 生成质量：low, medium, high
format string 否 输出格式：obj, fbx, glb 
curl -X POST "http://localhost:8091/api/generation/image-to-3d" \
  -F "image=@/path/to/image.jpg" \
  -F "quality=high" \
  -F "format=obj"
const formData = new FormData();
formData.append('image', fileInput.files[0]);
formData.append('quality', 'high');
formData.append('format', 'obj');

const response = await fetch('http://localhost:8091/api/generation/image-to-3d', {
  method: 'POST',
  body: formData
});

const result = await response.json();
/api/generation/task/{taskId}
GET
查询指定任务的生成状态和结果。可以实时获取任务进度、预计完成时间等信息。

路径参数

参数名 类型 必填 说明
taskId string 是 任务ID，由生成接口返回

响应示例

200 OK
{
  "code": "0000",
  "info": "success",
  "data": {
    "taskId": "task_123456789",
    "status": "completed",
    "progress": 100,
    "result": {
      "modelUrl": "https://example.com/models/cat_model.obj",
      "previewUrl": "https://example.com/previews/cat_preview.jpg",
      "downloadUrl": "https://example.com/downloads/cat_model.zip"
    },
    "createdAt": "2024-01-01T10:00:00Z",
    "completedAt": "2024-01-01T10:05:00Z"
  }
}
